# GBTI 3.1 · 股市人格鉴定中心

**不是你在炒股，是股市在定义你。**

GBTI（股市人格测试）是一个仿照 **MBTI / SBTI** 风格设计的娱乐向网页项目。  
它通过 **30 道题**，围绕用户在交易过程中的典型行为，从 **追涨倾向、消息依赖、止损能力、计划执行力** 四个核心维度进行识别，并最终生成一个专属的 **四字母股市人格代码**。

这不是一个严肃的金融测评工具，而是一套更适合社交传播的“**股市人格诊断系统**”——结合了人格测试的结构感、股民文化的梗感，以及 H5 页面式的视觉表达，适合分享到 **微信群、朋友圈、社交媒体** 等场景。

## 维度设计

GBTI 3.1 基于 4 组对立维度构建：

### 1. C / F
- **C = Calm 冷静派**：更倾向等待确认，不轻易被盘中波动带走
- **F = FOMO 追涨派**：更容易因为上涨、异动和情绪而冲动进场

### 2. L / R
- **L = Logic 逻辑派**：更相信分析、结构、验证
- **R = Rumor 消息派**：更容易被消息、题材、老师、群聊内容影响

### 3. K / T
- **K = Cut 止损派**：更愿意及时承认错误，控制亏损
- **T = Tank 死扛派**：更倾向继续持有，不愿轻易认亏

### 4. P / A
- **P = Plan 计划派**：更依赖预案、纪律和交易节奏
- **A = All-in 上头派**：更容易在情绪驱动下做出大动作

最终系统会生成类似以下的人格代码：

- `CLKP`
- `CRTA`
- `FLKA`
- `FRTA`
